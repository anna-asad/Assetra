# Assetra Product

## What is Assetra?
A web application to manage company assets (computers, furniture, equipment, etc.) from purchase to disposal.

## Who uses it?
- Admins (can see and manage everything)
- Managers (can see assets in their department)

## What problems does it solve?
- Tracks who has what asset
- Keeps a record of all asset changes
- Shows if assets are working or broken
- Calculates asset value over time
- Generates reports for audits

## What we're building in Iteration 1 (Weeks 1-2)
1. Login system (Admin and Manager roles)
2. Dashboard showing asset counts
3. Form to add new assets
4. Asset status tracking
5. Basic navigation menu
6. Database to store everything

## Technologies
- Node.js + Express (backend)
- HTML, CSS, JavaScript (frontend)
- SQL Server (database)
